NOTES:
======

1. The S*.M files (S1C, S1O, S2C, S2O, S3) are Simulink block diagram
   definitions exported as MATLAB functions. They contain graphical
   block positions and connections, not executable equations.

2. To fully implement the simulations, you need to:
   - Extract the differential equations from the Simulink blocks
   - Implement them using scipy.integrate.solve_ivp or similar
   - The key equations are in the 'Fcn' blocks of the Simulink models

3. Main equation blocks in Simulink models:
   - Qaxis/Daxis: Stator and rotor flux dynamics
   - Rotor: Mechanical dynamics (J*dw/dt = Te - Tmech)
   - abc2qds: Clarke transformation
   - qds2abc: Inverse Clarke transformation

4. Control nomenclature:
   - C suffix: Closed-loop (with speed feedback)
   - O suffix: Open-loop (V/f control without feedback)
   - 1, 2, 3: Project numbers with different control strategies

5. The Python files can be run to generate plots and initialize
   parameters, but the actual time-domain simulation requires
   implementing the ODE system from the Simulink models.

INDUCTION MOTOR MODEL:
======================

The models simulate a 20 HP (14.9 kW) three-phase induction motor using:
- Stationary reference frame (qd0) model
- 5 state variables: psi_qs, psi_ds, psi_qr', psi_dr', omega_r
- Variable frequency operation (V/f control or field-oriented control)

Motor parameters (from p20hp.py):
- Power: 20 HP (14920 W)
- Voltage: 220 V line-to-line
- Frequency: 60 Hz
- Poles: 4
- Rs = 0.1062 Ω
- Xls = 0.2145 Ω
- Xm = 5.8339 Ω
- Rr' = 0.0764 Ω
- Xlr' = 0.2145 Ω
- J = 2.8 kg·m²

SIMULATION SCENARIOS:
=====================

1. Load cycling at fixed frequency (M1C, M1O)
   - Start from standstill
   - Ramp up to rated speed
   - Apply load steps (rated load, half load, etc.)

2. Speed cycling at no-load (M1C, M1O)
   - Speed reversals
   - Positive to negative speed transitions

3. Field-oriented control (M3)
   - Decoupled flux and torque control
   - Dynamic torque response with constant flux
