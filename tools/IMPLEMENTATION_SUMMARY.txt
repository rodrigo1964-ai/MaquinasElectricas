C9 - INDUCTION MOTOR CONTROL
-----------------------------

1. S1C.py - V/f Closed Loop Control
   Location: /home/rodo/Maquinas/C9/S1C.py
   Description: Constant V/f ratio control with speed feedback
   Features:
   - Induction machine model in dq stationary frame
   - V/f ratio control: Vs/f = constant
   - PI speed controller with slip frequency calculation
   - Speed reference ramping (0 → 0.8 pu)
   - Step load torque application
   - Uses parameters from p20hp.py (20 HP machine)
   States: [psiqs, psids, psiqr, psidr, wr, theta, speed_int, i0s]
   Outputs: Speed tracking, torque, stator current magnitude

2. S1O.py - V/f Open Loop Control
   Location: /home/rodo/Maquinas/C9/S1O.py
   Description: Constant V/f ratio without speed feedback
   Features:
   - Open loop frequency command (1 Hz → 60 Hz ramp)
   - V/f proportional voltage control
   - Fan load (torque ∝ speed²)
   - Slip calculation and monitoring
   States: [psiqs, psids, psiqr, psidr, wr, theta, i0s]
   Outputs: Speed (rpm), frequency, slip (%), torque, current

3. S3.py - Field-Oriented Control (FOC)
   Location: /home/rodo/Maquinas/C9/S3.py
   Description: Rotor flux orientation with decoupled id*, iq* control
   Features:
   - Rotor flux vector orientation (d-axis aligned with flux)
   - Flux control (ids*) and torque control (iqs*)
   - Field weakening above base speed
   - PI controllers for speed and torque
   - Current regulation with P controllers
   - Step load disturbance rejection
   States: [psiqs, psids, psiqr, psidr, wr, theta_r, torque_int, i0s]
   Outputs: Speed tracking, rotor flux magnitude, torque, id/iq currents

INDUCTION MACHINE MODEL (Common to C9)
---------------------------------------
- 4th order model in stationary dq frame
- Stator flux linkages: ψqs, ψds
- Rotor flux linkages (rotor frame): ψqr, ψdr
- Mechanical dynamics: inertia (2H), damping (D)
- Electromagnetic torque: Te = ψqs*ids - ψds*iqs
- Parameters from p20hp.py:
  * 20 HP, 220V, 4-pole, 60 Hz
  * rs, xls, xm, rpr, xplr
  * H = 0.62 sec

C10 - POWER SYSTEMS
-------------------

4. S4.py - Power System Stabilizer (PSS)
   Location: /home/rodo/Maquinas/C10/S4.py
   Description: Synchronous generator with IEEE PSS
   Features:
   - IEEE Type 1 excitation system (AVR)
   - PSS with washout and two lead-lag compensators
   - Transient synchronous machine model (Edp, Eqp)
   - Step torque disturbance (10% increase at t=1s)
   - PSS enable/disable switch
   - Parameters from set1.py
   States: [Edp, Eqp, delta, slip, Ef, VR, Vs, wash_out, comp1_x, comp2_x]
   PSS Structure:
   - Washout: Tw = 5.0 s (high-pass filter)
   - Lead-lag 1: T1/T2 = 0.05/0.02
   - Lead-lag 2: T3/T4 = 3.0/5.4
   - Output limits: ±0.1 pu
   Outputs: Rotor angle, speed deviation, terminal voltage, power

5. S3.py - Subsynchronous Resonance (SSR)
   Location: /home/rodo/Maquinas/C10/S3.py
   Description: IEEE First Benchmark SSR model with torsional system
   Features:
   - 6-mass torsional system (HP, IP, LPA, LPB, GEN, EXC)
   - Detailed generator model with field and damper windings
   - Series capacitor compensation (45%)
   - Modal coordinate representation (Qbar transformation)
   - Natural frequencies: 99.78, 128.2, 161.1, 202.5, 303.4 rad/s
   - Network dynamics with series RL and capacitor
   - Parameters from i3essr.py (IEEE benchmark)
   States: [psid, psipf, psipkd, psipkq, psipkq2, psiq,
            delta_mode0, modal_angles[5], modal_speeds[5],
            vdc, vqc, idc, iqc] (21 states total)
   Outputs: Mode 0 angle, modal angles 1-5

KEY CONTROL CONCEPTS IMPLEMENTED
=================================

V/f Control (Scalar Control)
-----------------------------
- Maintains constant voltage-to-frequency ratio
- Prevents magnetic saturation at low frequencies
- Closed loop: uses speed feedback for slip control
- Open loop: frequency ramp command

Field-Oriented Control (Vector Control)
----------------------------------------
- Decouples flux and torque control
- Rotor flux orientation (d-axis aligned with rotor flux)
- ids* controls magnetizing flux
- iqs* controls electromagnetic torque
- Enables fast dynamic response

Power System Stabilizer
------------------------
- Provides supplementary damping control
- Input: speed deviation (Δω)
- Output: supplementary voltage signal to AVR
- Washout removes steady-state component
- Lead-lag compensators provide phase compensation
- Improves power system stability

Subsynchronous Resonance
-------------------------
- Interaction between turbine-generator torsional modes
  and series-compensated transmission system
- Can cause shaft fatigue or damage
- Modal analysis for multi-mass system
- Series capacitor creates subsynchronous frequencies
