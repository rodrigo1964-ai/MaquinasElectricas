## Equations Implemented

### Synchronous Machines (dq0 frame, 7 states)

```python
States: [δ, ψq, ψkq, ψd, ψf, ψkd, ωm]

# Stator flux derivatives
dψq/dt = vq + rs·iq - ωb·ωm·ψd
dψd/dt = vd + rs·id + ωb·ωm·ψq

# Field flux derivative
dψf/dt = vf - rpf·if

# Damper flux derivatives
dψkq/dt = -rpkq·ikq
dψkd/dt = -rpkd·ikd

# Mechanical
dωm/dt = (Tm - Te - D·Δω) / (2H)
dδ/dt = ωb·(ωm - 1)

# Torque
Te = ψd·iq - ψq·id

# Currents from flux linkages
ψaq = xMQ·(ψq/xls + ψkq/xplkq)
ψad = xMD·(ψd/xls + ψf/xplf + ψkd/xplkd)
iq = (ψq - ψaq) / xls
id = (ψd - ψad) / xls
if = (ψf - ψad) / xplf
ikq = (ψkq - ψaq) / xplkq
ikd = (ψkd - ψad) / xplkd
```

### PM Motor (6 states)

```python
States: [δ, ψq, ψkq, ψd, ψkd, ωslip]

# Similar to above but:
# - No field winding (PM excitation)
# - ψad includes PM: ψad = xMD·(ψd/xls + (ψkd + xmd·Ipm)/xplkd)
# - Slip dynamics: dωslip/dt = (Te - Tm) / (2H)
```

### DC Machines (2-3 states)

```python
# Separately Excited / Shunt (2 states: Ia, ωm)
dIa/dt = (Va - Ea - Ra·Ia) / La
dωm/dt = (Te - Tload - D·ωm) / J
Ea = Ka·ωm
Te = Ka·Ia

# Shunt Generator (3 states: If, Ia, ωm)
dIf/dt = (Vf - (Rf + Rrh)·If) / Lf
dIa/dt = (Ea - Ra·Ia - Va) / La
dωm/dt = (Tmech - Te - D·ωm) / J
Ea = f(If, ωm)  # from mag curve
Te = Ea·Ia / ωm
Vf = Va  # self-excited

# Series Motor (2 states: Ia, ωm)
dIa/dt = (Va - Ea - (Ra + Rse + Rext)·Ia) / (La + Lse)
dωm/dt = (Te - Tload) / J
Ea = f(Ia, ωm)  # from mag curve
Te = f(Ia)  # from mag curve
```

## Customization Quick Reference

### Change Disturbance Timing
```python
# Find arrays like:
Ex_time = np.array([0, 0.2, 0.2, 5])
# Modify timing or add more steps
```

### Change Disturbance Magnitude
```python
# Find arrays like:
Ex_value = np.array([Efo, Efo, 1.1*Efo, 1.1*Efo])
# Change 1.1 to desired multiplier
```

### Change Simulation Duration
```python
# Find:
t_stop = 5.0
# or
tstop = 5.0
# Change to desired seconds
```

### Change Initial Conditions
```python
# For sync machines, find:
P = 1.0  # Real power
Q = 0.0  # Reactive power

# For DC machines, find:
Ia0 = 0.0
wm0 = 0.0  # or wmrated
```

### Change Solver Accuracy
```python
# Find solve_ivp call, modify:
rtol=1e-6,  # Relative tolerance
atol=1e-8,  # Absolute tolerance
max_step=1e-3  # Maximum step size
```
